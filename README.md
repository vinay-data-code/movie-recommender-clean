
# 🎬 Movie Recommender System

A Netflix-style movie recommendation web app built using:

- Python
- Pandas
- Scikit-learn
- Streamlit
- TMDB API

## Features
- 🔍 Search movies
- 🎬 View movie details
- 🎭 Actor profiles with images
- 🎯 Get recommendations
- 🖼️ Posters and UI rendering
